League of Legends Win Predictor

Predicting league of legends results using machine learning

View on GitHub Download .zip Download .tar.gz

Aims

  • To train a machine learning application to successfully predict the winner of a League of Legends match based on the history of the match participants.
  • To create an application interface to interact with the system.

Method

  • A range of prediction methods will be trialed to see which is the most effective, including Neural Networks and Naive Bayes.
  • Information on participants will be collected using the API, including their win rate on the champion picked, weighted win rate on the champion picked based on number of games played, ranked win rate and weighted ranked win rate based on number of games played.
